# Player Score Calculator

## Description
This is a simple Python program that calculates the average score per game.

## Features
- Takes player name as input
- Accepts number of games played
- Accepts total score
- Calculates average score using division

## How to Run

1. Open terminal
2. Navigate to project folder
3. Run the command:

py game_score.py

Games Played: 5  
Total Score: 250  
Average Score: 50.0
